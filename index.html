    Q1:What is the use of clear property in css

    Usage of clear property

    • The clear property controls the flow next to floated elements.
    • The clear property specifies what should happen with the element that is next to a floating element.
    • When you float an element, it can cause other elements to wrap around it, which may not always be the desired behavior. The clear property allows you to specify whether an element should be allowed to float alongside previously floated elements or if it should be forced below them.
    • clear property values:
      1. none
      2. left
      3. right
      4. both
    • Example:

      • This text is wrap around html logo using float property value left.

      On this text the clear propert value left is used which work as an anti-float.
